🚀 완전한 UI/UX 구현 계획: 9단계 환자 여정 + 관리자 시스템
목표: 부드러운 애니메이션, 명확한 플로우, 깔끔한 구조, 고령자 친화적, 충돌 없음, 직관적 UI를 모두 만족하는 완성도 높은 병원 안내 시스템

기존에 있는 파일부터 분석한 뒤, 아래 내용을 실행해야 함.

📱 전체 화면 구조 및 라우팅
A. 비로그인 사용자
🔊 VoiceGuideScreen (/voice-guide)
├── 음성 인식 버튼 (대형)
├── 말풍선 애니메이션으로 "무엇을 도와드릴까요?" 
├── 자주 묻는 질문 버튼들
└── 화면 하단: "로그인하면 더 많은 정보를 확인할 수 있어요"
B. 환자 사용자 (9단계)
🏠 HomeScreen (/) - 상태별 동적 화면
├── UNREGISTERED → UnregisteredScreen
├── ARRIVED → ArrivedScreen  
├── REGISTERED → RegisteredScreen
├── WAITING → WaitingScreen
├── CALLED → CalledScreen
├── ONGOING → OngoingScreen
├── COMPLETED → CompletedScreen
├── PAYMENT → PaymentScreen
└── FINISHED → FinishedScreen

📋 ProfileScreen (/profile)
├── 내 정보 수정
├── 로그아웃
└── 앱 설정

🔊 VoiceScreen (/voice) - 로그인 후 음성 안내
├── 개인화된 음성 안내
└── 내 상황에 맞는 답변
C. 관리자 사용자
🏢 AdminHomeScreen (/admin)
├── 오늘의 병원 현황 요약
├── 긴급 알림 목록
└── "상세 관리하기" 버튼 → 기존 대시보드로 이동

📊 AdminDashboard (/dashboard) - 기존 구현된 대시보드
├── NFCTagManagement.jsx
├── QueueMonitoring.jsx  
├── AnalyticsDashboard.jsx
└── AdminDashboard.jsx

🔊 음성 안내 시스템
VoiceGuideScreen (비로그인)
jsx
주요 기능:
🎤 대형 음성 인식 버튼 (파동 애니메이션)
💬 "무엇을 도와드릴까요?" 말풍선
🔥 인기 질문 버튼들
├── "화장실 어디에 있나요?"
├── "주차장 요금은 얼마인가요?"  
├── "MRI 검사 시간 얼마나 걸리나요?"
└── "원무과는 어디인가요?"

음성 인식 플로우:
1. 버튼 터치 → 마이크 활성화 (파동 애니메이션)
2. 음성 인식 중 → 실시간 텍스트 표시
3. 인식 완료 → 답변 텍스트 + 음성 출력
4. 관련 추가 질문 버튼 제안
